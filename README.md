# 沙盘仪表盘 - 产品矩阵分析工具

基于 Web 的产品矩阵可视化分析工具，支持从 Excel 导入数据，生成销量-价格矩阵沙盘，并支持跨设备分享。

## 功能特性

- **双页面结构**：京东竞品沙盘 + 电商新品跟踪
- **数据导入**：支持 Excel (.xlsx) 文件上传和 API 加载
- **分享功能**：链接分享（同一浏览器）+ JSON 导出（跨设备）
- **数据可视化**：销量-价格矩阵布局，品牌颜色标识，环比趋势分析

## 文件说明

| 文件 | 说明 |
|------|------|
| `index.html` | 入口页面，自动跳转到编辑页面 |
| `沙盘仪表盘-带分享.html` | 编辑页面，支持数据上传、查看、分享 |
| `沙盘查看.html` | 查看页面，用于打开分享的链接 |

## 快速开始

1. 直接在浏览器中打开 `沙盘仪表盘-带分享.html`
2. 上传 Excel 文件或从 API 加载数据
3. 查看生成的沙盘矩阵
4. 点击"分享沙盘"导出 JSON 文件分享给他人

## 使用说明

### 编辑页面（沙盘仪表盘-带分享.html）

1. **上传数据**：点击"上传 Excel"按钮选择文件，或点击"从API加载"配置API
2. **选择月份**：在月份选择器中选择要查看的月份
3. **筛选品牌**：点击"选择品牌"按钮筛选要显示的品牌
4. **切换分类**：点击六大赛道标签切换不同产品类别
5. **分享数据**：点击"分享沙盘"按钮，选择链接分享或导出JSON文件

### 查看页面（沙盘查看.html）

1. **打开分享链接**：直接访问分享的链接查看数据
2. **导入JSON文件**：如果显示"无法加载沙盘数据"，点击"选择 JSON 文件"导入

## Excel 格式要求

### 京东竞品沙盘（第1个sheet）

**必需列：**
- 日期、产品型号、重点品牌、单价、成交数量、产品大类

**可选列：**
- 二级品牌、产品图片链接、分辨率、商品链接

### 电商新品跟踪（第2个sheet）

**必需列：**
- 区域、商品SKU（ASIN）、商品链接、销量日期、上架时间、图片链接、销量、型号、品牌、单价、产品大类、分辨率

## GitHub Pages 部署

1. 上传所有文件到 GitHub 仓库
2. 进入 Settings → Pages
3. Source 选择 "Deploy from a branch"
4. Branch 选择 "main" 和 "/ (root)"
5. 点击 Save，等待部署完成
6. 访问 `https://你的用户名.github.io/仓库名/`
